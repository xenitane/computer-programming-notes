These operators are used to do basic Arithmetic Operations on the data we are presented with.

These are the Arithmetic operators available to us
1. **+** : Addition
2. **-** :Subtraction
3. **\*** : Multiplication
4. **\/** : Division or Quotient
5. **%** : Modulo or Remainder
6. Parenthesis **( )**

## Examples
```c
int a=10, b=3;

a+b          // 13
a-b          // 7
a*b          // 30
a/b          // 3
a%b          // 1
(a+b)/(b*b)  // 1

float x=2.5, y=4.0;

x/y          // 0.625
```
